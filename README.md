# IPL Dashboard (2008 - 2022)

An interactive Power BI dashboard designed to analyze Indian Premier League (IPL) data from 2008 to 2022. This project provides deep insights into match outcomes, player performances, and team statistics.

## 📊 Features
* **Match Analysis:** Filter matches by season, city, and winner.
* **Toss Impact:** Visual representation of how winning the toss affects match results.
* **Player Performance:** Key metrics for "Man of the Match" awards and top performers.
* **Venue Stats:** Analysis of match results based on different stadiums.
* **Winning Margins:** Breakdown of wins by runs vs. wins by wickets.

## 🛠️ Tools Used
* **Power BI Desktop:** For data visualization and dashboard creation.
* **MS Excel / CSV:** Source data handling.
* **DAX (Data Analysis Expressions):** Used for creating custom measures and calculated columns.

## 📂 Project Structure
* `iPL DASHBOARD.pbix`: The main Power BI project file.
* `ipl_matches_2008_2022.csv`: The raw dataset used for analysis.
* `IPL-Stadium-Background-Template-edit-online.jpg`: Custom background used for the dashboard UI.

## 🚀 How to View
1. Download the `iPL DASHBOARD.pbix` file.
2. Install [Power BI Desktop](https://powerbi.microsoft.com/desktop/).
3. Open the file to interact with the visualizations.
